# ProgrammingLanguage
This repo is about understand what a programming language mean and it's constructs.

Programming language deals with Data and Operation. 
Data includes
- data type.
- data structures.
- data conversion.
- Data Representation (how data is stored in memory) 

Operation includes
- Comparision
- loop over data
- functions
- methods
